# 李胤儒 Personal Research Website

Interactive personal research homepage focused on:

- UAV-based inspection
- Computer vision
- Transportation infrastructure safety

The site is built with Next.js App Router, TypeScript, Tailwind CSS, local data files, and static assets. It does not use a database, login system, CMS, or contact-form backend.

## Run Locally

```bash
npm run dev
```

Open `http://127.0.0.1:3000`.

## Verify

```bash
npm run lint
npm run build
```

On this Windows sandbox, `next build` may require permission to spawn worker processes. The source itself builds successfully once worker spawning is allowed.
